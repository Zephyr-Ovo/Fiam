# 控制台

> SvelteKit 5 Web 仪表盘：记忆图谱可视化、事件管理、beat 流查看。

## 技术栈

- **SvelteKit 2.57 + Svelte 5** (runes)
- **Tailwind CSS 4** + Catppuccin palette
- **Canvas 2D** 自写力导向图谱渲染（从 Cytoscape.js 迁移而来）
- 纯静态构建 → dashboard_server.py 通过 Flask serve

### 为什么不用 Cytoscape？

最初用了 Cytoscape.js (`63aaab4`)。但 Zephyr 想要更精细的控制——脑形约束、3D 轨道旋转、节点拖拽——Cytoscape 的定制化不够灵活。改成自己用 Canvas 2D 画，力导向布局自己写。

## 页面

### Graph 页 — 实时记忆图谱

`dashboard/src/routes/graph/`

力导向图，节点 = 事件，边 = 关系。

特性：
- **脑形软椭球约束** — 节点被约束在一个椭球体内，看起来像大脑形状（`d5dcdea`）
- **3D 轨道旋转** — 鼠标拖拽旋转视角；缩放
- **节点拖拽** — 拖动节点，力导向实时重新布局
- **悬停提示** — 显示事件名称
- **边悬停** — 显示边类型 + 权重（`167d472`）
- **右键菜单** — 边类型/权重编辑（`EdgeMenu` 组件）
- **NodeEditor** — 点击节点 → 侧边栏显示事件详情，支持编辑
- **删除** — 两次点击确认删除，执行级联删除（[[Pool 存储设计#级联删除]]）
- **自动漂移** — 去掉了旋转动画，换成缓慢的 sin/cos pan 漂移
- **Recall sparkle** — 记忆被激活时播放五音正弦波衰减音（1200-2400Hz）

### Events 页 — 事件列表

事件按时间排序。点击进入详情：body 文本、元数据、关联边。

### Flow 页 — Beat 流

实时显示 flow.jsonl 的内容。每条 beat 显示时间、来源标签、文本内容。可以看到信息流动的实时全景。

### Schedule 页 — 定时任务

显示当前排定的 WAKE 任务，以及历史执行记录。

### Logs 页 — 日志

daemon 运行日志、成本记录。

## 后端

`scripts/dashboard_server.py` — Flask app，监听 :8766（127.0.0.1 only）。

### API

Pool 相关：
- `GET /api/pool/events` — 事件列表
- `GET /api/pool/event/<id>` — 事件详情
- `POST /api/pool/event/delete/<id>` — 删除事件（级联）
- `GET /api/pool/graph` — 图数据（节点 + 边）

Legacy（过渡期保留）:
- `GET /api/graph` — 旧格式图数据

通用：
- `POST /api/capture` — Favilla/Limen 数据摄入
- `GET /api/flow` — beat 流数据
- `GET /api/daemon_state` — daemon 运行状态

### 认证

两级 token：
- `FIAM_VIEW_TOKEN` — 读操作（查看图谱、事件、日志）
- `FIAM_INGEST_TOKEN` — 写操作（capture、编辑、删除）

### 静态文件

SvelteKit 构建产物在 `dashboard/build/`，server 做 SPA fallback（所有非 API 路径返回 index.html）。

```python
# 构建：
cd dashboard && npm run build

# ISP 部署：
ssh isp "~/start_dashboard.sh"  # pkill + nohup setsid
```

## 主题

深暗蓝色基底 `#0c0c1a`，Catppuccin palette 点缀。字体：Anthropic Sans / Serif（本地 woff2）。

缩放级别 ≥ 2 时显示 graph legend。低缩放级别保持画面干净。
