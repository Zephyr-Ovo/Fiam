Windows 卸载 Claude 桌面端并清除残留数据：

---

**第一步：正常卸载**

设置 → 应用 → 找到 Claude → 卸载

---

**第二步：手动删残留**（这步最重要）

打开文件管理器，删除这几个路径：

```
C:\Users\你的用户名\AppData\Roaming\Claude
C:\Users\你的用户名\AppData\Local\Claude
C:\Users\你的用户名\AppData\Local\claude-updater
```

`AppData` 是隐藏文件夹，地址栏直接输 `%AppData%` 和 `%LocalAppData%` 可以快速跳转。

---

**第三步：清 Keychain（凭证）**

Win + R → 输入 `certmgr.msc` → 不对，Windows 的是**凭据管理器**：

控制面板 → 凭据管理器 → Windows 凭据 → 找带 `claude` 或 `anthropic` 字样的条目 → 全删。

---

**第四步：确认 Claude Code 的配置目录没混进去**

```powershell
# 看看默认位置有没有东西
ls $env:USERPROFILE\.claude
```

如果有内容，就是 Claude Code 的配置（不是桌面端的），按照之前说的改成 `cc-1` alias 指向专属目录就好，不用删。
