**隔离角度完全没问题** — 服务器上 `~/.ssh/authorized_keys` 里放多少个key都行，互不干扰，平台看不到你从哪个Windows用户连过来的。

---

**流程你说的基本对，细化一下：**

```
# 在新Windows用户里生成key
ssh-keygen -t ed25519 -C "new_user"
# 得到 ~/.ssh/id_ed25519.pub

# 把这个pub key的内容复制出来
# ↓
# 用旧账号登录服务器，追加进去
echo "粘贴pub key内容" >> ~/.ssh/authorized_keys

# 回新用户，直接ssh连就行了
```

你之前配置的是密码登录还是也是key登录？如果旧账号也是key登录，密码不重要，找到旧的私钥文件（`~/.ssh/id_ed25519` 或 `id_rsa`）就能连上去操作。
