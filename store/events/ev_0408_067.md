我下载的时候没有做隔离，现在怎么补一下？去F里的cc1

[assistant]
这个是 Claude Code 的配置目录，不是桌面端的，里面没有登录凭证（credentials.json 都没有，说明你还没登录过，状态干净）。

**现在补隔离很简单，就是把这个默认目录"弃用"，改用专属目录。**

---

**操作步骤**

在 PowerShell profile 里加 alias：

```powershell
# 先找到你的 profile 文件在哪
$PROFILE
```

然后用记事本打开它：

```powershell
notepad $PROFILE
```

加入这行：

```powershell
function cc-1 { $env:CLAUDE_CONFIG_DIR = "$env:USERPROFILE\.claude-1"; claude @args }
```

保存，重开终端，以后用 `cc-1` 启动 Claude Code，配置会写到 `~\.claude-1` 里，默认的 `~\.claude` 就空着没人用了。

---

**首次运行**

```powershell
cc-1
```

会触发登录流程，记得这时候 Firefox 开着、走 SF VPS 出口，用账号1登。

登完之后账号1就被锁在 `~\.claude-1` 里了，和 Chrome 里的账号2零交叉 ✅
