看了一眼你的截图和代码——

那两条线是因为你同时有：

1. **CSS 原生的 `text-decoration: underline`**（浏览器默认给链接加的下划线）
2. **你自己加的 `border-bottom: 0.5px solid var(--claude-accent-dim)`**

所以叠了两条。

修复方法，在你的链接样式里加上 `text-decoration: none` 并去掉 `border-bottom`：

```css
a, .cm-hmd-internal-link, .cm-link {
  color: var(--link-color);
  text-decoration: none;
  /* 删掉这行 → border-bottom: 0.5px solid var(--claude-accent-dim); */
  transition: color 0.15s;
}
a:hover {
  color: var(--link-color-hover);
  /* 删掉这行 → border-bottom-color: var(--claude-accent); */
}
```

你的代码里其实已经有 `text-decoration: none` 了，所以只需要**把 `border-bottom` 那两行删掉**就行，两条线就都没了。
